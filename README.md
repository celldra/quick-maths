# quick-maths
Simple mental maths game written in Rust

## Compliation
You'll need the Rust stable (or nightly) toolchain to compile.

1. Clone the repository

```sh
git clone https://github.com/celldra/quick-maths.git
```

2. Compile

```sh
cd quick-maths
cargo build --release
```

3. Run

```sh
cd target/release
./quick-maths
```
*Profit!*
